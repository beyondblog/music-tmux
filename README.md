music-tmux
==========

Music player in terminal


## Build

For linux or MacOS

```
% yum install libmpg libao libconfig

or

% apt-get install libmpg libao libconfig

MacOS 

% brew install mpg libao libconfig

% make

% make install

```

## Run

```

% music_tmux

```

## Config file

~/.music_tmux.conf
